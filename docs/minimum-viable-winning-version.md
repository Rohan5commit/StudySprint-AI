# Minimum Viable Winning Version

If time gets tight, keep only this vertical slice:

## Must keep
- polished landing page
- one onboarding form
- one seeded sample subject
- generated summary
- chapter checklist
- quiz mode
- flashcards
- revision timetable
- last-minute revision mode card

## Can simplify
- only one subject flow in the live demo
- local state only
- no auth
- no database
- no file parsing beyond pasted text
- last-minute mode can reuse existing generated content

## Winning angle
Even in minimum form, the product still feels complete because the full student story is visible end to end.
