# v0 Prompt

Build a polished student web app called **StudyPilot AI**.

Product goal:
A student enters their subject, chapter/topic, exam date, weak chapters, available study hours per week, and learning style. The app generates a personalized study system.

Required screens:
1. Landing page
2. Onboarding / workspace form
3. Dashboard
4. Study plan view
5. Quiz mode
6. Flashcard mode

Visual style:
- clean premium student productivity product
- dark modern SaaS aesthetic
- no generic purple AI gradients
- strong spacing, rounded cards, clear hierarchy
- very demo-friendly and easy for judges to understand fast

Required UI elements:
- hero section with one-line value proposition
- CTA buttons: “Launch judge demo” and “Start onboarding”
- onboarding fields for subject, chapter/topic, exam date, study hours per week, weak chapters/topics, learning style, and syllabus/notes text
- dashboard cards for checklist progress, quiz accuracy, flashcards mastery, and exam runway
- summary section
- chapter checklist section
- weak-topic highlight chips
- revision timetable cards
- quiz question cards with multiple-choice answers
- flashcards with flip interaction
- last-minute revision mode card

Tone:
practical, student-friendly, useful, not gimmicky.
